# DOM-Project: Productivity Hub

A modern, interactive web application designed to help users manage their daily tasks and productivity. Built with vanilla JavaScript, HTML, and CSS, this project demonstrates DOM manipulation and event-driven architecture.

## 🎯 Overview

DOM-Project is a comprehensive productivity tool that combines multiple features in a single, user-friendly interface. It provides an all-in-one solution for task management, time tracking, and daily planning.

## ✨ Features

- **Todo List** - Create, manage, and track your daily tasks with seamless user interactions
- **Daily Planner** - Organize your schedule and plan your day effectively
- **Motivation Quotes** - Get inspired with motivational quotes throughout your day
- **Pomodoro Timer** - Boost productivity with the scientifically-proven Pomodoro technique for focused work sessions
- **Daily Goals** - Set and track your personal daily objectives

## 🛠️ Technologies Used

- **Frontend:**
  - HTML5 - Semantic structure and accessibility
  - CSS3 - Custom properties (CSS variables), Flexbox layout, responsive design
  - Vanilla JavaScript - DOM manipulation, event handling, and interactive features
- **Design:**
  - Remix Icon - Intuitive icon library for enhanced UI
  - Custom color scheme with CSS variables for maintainability

## 📋 Project Structure

## 🚀 How to Use

1. **Clone or download** the project files to your local machine
2. **Open `index.html`** in your web browser
3. **Browse features** by clicking on any of the five available options:
   - Click on the feature card to open the full-page view
   - Click the "Close" button to return to the main menu
4. **Use the tools** to manage your tasks and productivity
